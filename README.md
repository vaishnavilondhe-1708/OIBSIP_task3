## Overview
This project focuses on data cleaning and preprocessing of the Airbnb New York City 2019 dataset as part of an internship task. The objective is to ensure data quality and prepare the dataset for accurate analysis and modeling.

- Duplicate record removal  
- Feature standardization

## Tools Used

- Google Colab  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Outcome
A clean, consistent, and analysis-ready dataset that ensures reliable insights and improved model performance.
